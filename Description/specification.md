Описание:
На предприятии налажен выпуск персональных компьютеров. 
Для сборки используются комплектующие различных фирм-производителей.
Стоимость аналогичных компонентов от разных фирм может быть разной. 

Модель выпускаемых компьютеров - это компьютер с определенными характеристиками.
В зависимости от партии, в модели компьютера используются комплектующие разных фирм-производителей. 

Стоимость модели компьютера для потребителя фиксирована. 
Себестоимость одной модели может отличаться по в зависимости от партии, т.к. определяется по стоимости использованных комплектующих. 

На чистую прибыль компании оказывает влияние фактический срок службы компонентов компьютера. При выходе из строя любого компонента компьютера, изделие бракуется целиком, и его полная стоимость возвращается покупателю. 

Для оптимизации прибыли в компании было решено создать хранилище данных, в который будет поступать информация о продажах и возвратах. 
На основании данных из этого хранилища необходимо будет получать отчет о соотношении прибыль/срок службы в разрезе типов комплектующих и их производителей.

Задача:
Необходимо разработать модель хранилища данных и написать запрос для построения данного отчета.
